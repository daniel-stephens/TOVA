How to use TOVA

Follow the same order as the shortcuts on the home page and in the top navigation: upload, corpus library, training, then trained models.

  1. 1. Upload Data Add documents and validate them so they are ready for preprocessing and training jobs.
  2. 2. Corpus library Review saved corpora, versions, and metadata. Pick a corpus when you start training.
  3. 3. Initiate Training Choose a classical or LLM-assisted pipeline, set parameters, and start a training run. Monitor progress from the job view.
  4. 4. Trained topic models Open a completed model for dashboards, theme summaries, document tables, and exploration.

LLM configuration When signed in, open your account menu (top right) and choose LLM settings to pick providers, models, and API options used for LLM-assisted training, topic labeling, summarization, and chat.

Global app configuration In the same menu, Config editor opens global defaults (preprocessing, classical topic modeling, and related YAML-backed options) stored with your account.
